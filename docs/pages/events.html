Events

Random events add unpredictability and excitement to your scenario. Each turn, the server rolls for every registered event and fires it if the dice land in range.

How Random Events Work

Every turn (step), the server iterates through all events defined in your scenario. For each event it generates a random number between 0.0 and 1.0. If the roll is less than the event's chance value, and all other constraints are met, the event fires and its effect callback is executed.

s.event("dragon_attack", {
  chance: 0.02,
  minStep: 10,
  maxPerGame: 3,
  message: "A wild dragon descends from the mountains!",
  effect(e) {
    const cell = e.randomCell({ biome: "mountain" });
    e.spawnNeutral("dragon", cell);
    e.broadcast("Beware — a dragon has appeared!");
  }
});

Event Parameters

Parameter Type Description
chance number Probability of firing per step, from 0.0 to 1.0. Typical range: 0.005–0.05.
minStep number The event will never fire before this turn number. Use this to prevent early-game disruption.
maxPerGame number Maximum total times this event can fire in a single game. Omit for unlimited.
message string Broadcast text sent to all players when the event fires.
effect function Callback that receives an effect object e with methods to modify the game world.

Effect Callback Object (e)

The effect function receives a single argument — the effect callback object. This object exposes every method you need to modify the game world when an event fires.

Spawning

Method Description
e.spawnNeutral(type, cell) Spawn a single neutral (unowned) unit of the given type at the specified cell.
e.spawnNeutralGroup(type, count, cell?) Spawn multiple neutral units of the same type. If cell is omitted, units are spread across random cells.

Cell Selection

Method Description
e.randomCell({ biome? }) Return a single random cell. Optionally filter by biome name.
e.randomCells(count, { biome? }) Return an array of count random cells. Optionally filter by biome.
e.allCells({ biome? }) Return all cells on the map. Optionally filter by biome.

Resources & Overlays

Method Description
e.addResource(cell, type, amount) Add a resource deposit to a cell (e.g., gold, scrap).
e.addOverlay(cell, overlayName, { duration? }) Apply a visual/functional overlay to a cell. Optional duration in turns — omit for permanent.

Units & Status

Method Description
e.allUnits() Return an array of every unit currently on the map (all players and neutrals).
e.applyStatus(unit, effectName) Apply a named status effect to a unit (e.g., "poisoned", "irradiated").

Global & Communication

Method Description
e.startGlobalEvent(name, opts) Trigger a named global event with custom options (e.g., a map-wide storm with a duration).
e.broadcast(message) Send a text message to all players. Use for narrative flavor or warnings.
e.giveAllPlayers(resource, amount) Grant every player a specified amount of a resource.

Probability Math

The chance parameter is a simple per-turn probability. Some useful reference points:

chance Per-turn probability Expected frequency
0.005 0.5% ~1 per 200 turns
0.01 1% ~1 per 100 turns
0.02 2% ~1 per 50 turns
0.05 5% ~1 per 20 turns

A chance of 0.02 means roughly 2% per turn, so you can expect the event to fire approximately once every 50 turns. Over a 100-turn game, that's about 2 occurrences on average.

Event Design Tips

  • Balance disruption vs. fun. Events should shake up the game without making players feel helpless. Pair destructive events (dragon attacks, plagues) with beneficial ones (gold rushes, holy ground).
  • Use minStep generously. Early turns are critical for establishing a base. Avoid firing dangerous events before players can defend themselves.
  • Cap with maxPerGame. Even rare events can stack up in long games. Limit devastating events to 2–3 occurrences max.
  • Broadcast context. Always include a descriptive message so players understand what happened and why their map just changed.
  • Mix scales. Some events should be local (one cell), some regional (a cluster), and some global (affecting all players).

Castle: Age of Kingdoms Examples

// Dragon Attack — a dragon spawns in the mountains
s.event("dragon_attack", {
  chance: 0.02,
  minStep: 15,
  maxPerGame: 3,
  message: "A fearsome dragon emerges from a volcanic peak!",
  effect(e) {
    const cell = e.randomCell({ biome: "mountain" });
    e.spawnNeutral("dragon", cell);
  }
});

// Undead Uprising — skeletons rise across the map
s.event("undead_uprising", {
  chance: 0.015,
  minStep: 20,
  maxPerGame: 2,
  message: "The dead are rising from their graves!",
  effect(e) {
    e.spawnNeutralGroup("skeleton", 6);
  }
});

// Gold Rush — bonus gold deposits appear
s.event("gold_rush", {
  chance: 0.03,
  minStep: 5,
  maxPerGame: 4,
  message: "Gold veins have been discovered across the land!",
  effect(e) {
    const cells = e.randomCells(5, { biome: "mountain" });
    for (const cell of cells) {
      e.addResource(cell, "gold", 50);
    }
    e.giveAllPlayers("gold", 25);
  }
});

// Plague — all units risk being poisoned
s.event("plague", {
  chance: 0.01,
  minStep: 25,
  maxPerGame: 2,
  message: "A terrible plague sweeps across the kingdom!",
  effect(e) {
    for (const unit of e.allUnits()) {
      e.applyStatus(unit, "poisoned");
    }
  }
});

// Magical Storm — overlay disrupts a region
s.event("magical_storm", {
  chance: 0.025,
  minStep: 10,
  maxPerGame: 5,
  message: "A magical storm crackles across the horizon!",
  effect(e) {
    const cells = e.randomCells(8, { biome: "plains" });
    for (const cell of cells) {
      e.addOverlay(cell, "storm", { duration: 3 });
    }
    e.startGlobalEvent("storm_season", { duration: 3 });
  }
});

// Holy Ground — blessed cells boost nearby units
s.event("holy_ground", {
  chance: 0.02,
  minStep: 8,
  maxPerGame: 3,
  message: "Sacred light bathes the land — holy ground has appeared!",
  effect(e) {
    const cells = e.randomCells(3);
    for (const cell of cells) {
      e.addOverlay(cell, "holy_ground");
    }
  }
});

// Tournament — all players receive bonus gold
s.event("tournament", {
  chance: 0.03,
  minStep: 12,
  maxPerGame: 4,
  message: "A grand tournament is held! All kingdoms prosper.",
  effect(e) {
    e.giveAllPlayers("gold", 75);
    e.broadcast("The tournament rewards flow into every treasury.");
  }
});
